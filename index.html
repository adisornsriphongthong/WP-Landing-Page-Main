java.sql.Statement statement = connection.createStatement();: This line creates a Statement object that can be used to execute SQL statements against the database. ResultSet resultSet = statement.executeQuery(sqlQuery); This line executes the SQL query stored in the sqlQuery string and retrieves the results into a ResultSet object.
